JavaScript es un lenguaje de programación ampliamente usado para añadir interactividad y dinamismo a las páginas web. Permite realizar tareas como responder a eventos del usuario, modificar el contenido dinámicamente o interactuar con APIs externas.

JavaScript interactúa con el DOM (Document Object Model), una representación en árbol de los elementos HTML de una página web. Puede leer, modificar y responder a eventos en los elementos HTML, permitiendo cambios dinámicos en el contenido y estilo.

Un ejemplo es usar JavaScript para cambiar contenido en respuesta a eventos: 
                    <script>
                      document.getElementById('boton').addEventListener('click', function() {
                        alert('¡Hola, mundo interactivo!');
                      });
                    </script>

Git es un sistema de control de versiones que permite rastrear cambios en el código, colaborar en equipo y revertir errores. Es esencial para gestionar versiones y coordinar esfuerzos en proyectos de desarrollo web.

Git permite a los desarrolladores:
  • Hacer seguimiento de cambios en archivos.
  • Revertir a versiones anteriores en caso de errores.
  • Trabajar en paralelo en diferentes funcionalidades utilizando ramas.
  • Fusionar cambios de varias ramas en un proyecto principal.

GitHub es una plataforma basada en Git que permite alojar repositorios de código en la nube. Facilita la colaboración gracias a herramientas como pull requests, seguimiento de problemas y repositorios remotos.

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para estructurar y presentar contenido en la web. Su función principal es organizar el contenido de una página web mediante elementos que definen títulos, párrafos, imágenes, enlaces, tablas y más.

Las etiquetas HTML son elementos que marcan el inicio y el final de un contenido dentro de la página web. Generalmente vienen en pares, como <p> y </p>.

CSS (Cascading Style Sheets) es el lenguaje que controla el diseño y la presentación de una página web. Se usa para definir colores, fuentes, márgenes y más, brindando estilo y atractivo a las páginas HTML.

Algunas propiedades importantes de CSS incluyen:
  • color: Cambia el color del texto.
  • font-size: Ajusta el tamaño de la fuente.
  • margin: Controla el espacio exterior de un elemento.
  • padding: Ajusta el espacio interior de un elemento.
  • background-color: Cambia el color de fondo.

El modelo de caja describe las dimensiones de un elemento, compuesto de:
  • Contenido: El área donde se coloca texto o imágenes.
  • Padding: Espacio entre el contenido y el borde.
  • Border: La línea que rodea al padding.
  • Margin: Espacio exterior que separa el elemento de otros.

El diseño responsivo adapta las páginas web a diferentes tamaños de pantalla (móviles, tabletas, computadoras). Se logra mediante:
  • Media queries (@media).
  • Unidades relativas como %, em, rem, vw, y vh.
  • Herramientas de diseño flexible como Flexbox y Grid Layout.
  • Imágenes fluidas ajustadas con max-width: 100%;.
